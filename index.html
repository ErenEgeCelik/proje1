Explore the Solar System

Welcome to the Interactive Orrery! This application will help you explore celestial bodies in our solar system.Click on the planets to get more information about them.

About This Model

This solar system simulation was created using Three.js, a JavaScript 3D library, to help visualize the orbits and properties of celestial bodies in our solar system. The planetary data, including the size and distance from the Sun, was sourced from NASA's publicly available datasets.
